1) Run the server as it is and notice what is written to the command window as you click on the links. What is going on here? (Short paragraph or two.)

-- First of all when the server is ran, it gives you the time and status that the server started. Now clicking links it first shows you the address(client) from where the request is made then followed by date/time and type of request(Get). It then follows by the what it is looking for with the version of http using. Followed by 200 response status code which means all is good. Things happened as how the client wanted i.e Link to the file is found, request was received and is succeeded.
